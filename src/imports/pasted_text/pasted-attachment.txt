SECONDARY:       Compression vests, shorts, full sets — as catalog expands

## PART II — VISUAL IDENTITY SYSTEM

### 2.1 Color Architecture

```
PALETTE PHILOSOPHY:
Dark, high-contrast, gym-aesthetic. Like a premium gym in low light.
Black and deep surfaces dominate — making the human body (and the product on it)
the only real color on the page. Products stand out against dark backgrounds
the same way a physique stands out under gym lighting.

PRIMARY SURFACES:
  Gym Black:         #0C0C0C   (primary background — deep, warm black)
  Surface Elevated:  #141414   (card backgrounds, panels)
  Surface Mid:       #1C1C1C   (secondary panels, input backgrounds)
  Border Subtle:     #2A2A2A   (all dividers, borders)

TEXT HIERARCHY:
  White Pure:        #F2F2F2   (primary text — warm off-white, not pure)
  Gray Light:        #A0A0A0   (secondary text, captions, specs)
  Gray Mid:          #606060   (muted info, disabled)

BRAND ACCENTS:
  Cotton Blue:       #4A7BF7   (primary brand accent — CTAs, active states,
                               key highlights — blue has performance/tech coding)
  Electric White:    #FFFFFF   (high-contrast text moments, logo on dark)
  Performance Red:   #E03030   (urgency — sold out, limited stock, flash sale)
  Warm Khaki:        #C9A96E   (optional secondary — for premium colourway/product
                               moments — gives the "cotton" warmth the name implies)
  Gym Concrete:      #3A3A3A   (product card neutral backgrounds)

USAGE RULES:
  — 75% of screen: Gym Black or Surface Elevated
  — Cotton Blue: CTAs, active states, accent elements ONLY — not backgrounds
  — Performance Red: urgency and scarcity ONLY — never decorative
  — All product images: shot on DARK backgrounds or on-body (not white studio)
  — Photography: high-contrast, slightly desaturated, gym-light treatment

NOTE ON THE BRAND NAME "COTTON AURA":
  Despite the name suggesting softness, the visual identity is performance-dark.
  The name is the warmth signal — the product delivers toughness.
  This tension (soft name, hard aesthetic) is the brand's personality.
```

### 2.2 Typography System

```
DISPLAY / PERFORMANCE FONT — "Druk" or "Bebas Neue" or "Barlow Condensed Bold"
  Character: Condensed, powerful, athletic — the typography of jersey numbers
             and sports editorial. Commands space without filling it.
  Use: Hero headlines, collection names, campaign statements, section titles,
       product category headers, sale/urgency banners
  Weight: Bold / Extra Bold only
  Sizes:
    Hero H1:          clamp(64px, 12vw, 160px)
    Section H2:       clamp(40px, 6vw, 96px)
    Feature callout:  clamp(28px, 4vw, 60px)
    Category label:   clamp(20px, 2.5vw, 36px)
  Letter-spacing: -0.01em to 0em
  Line-height: 0.88–0.95 (compressed, athletic stacking)
  Transform: ALL CAPS always for display type

BODY / UI FONT — "DM Sans" or "Inter" or "Neue Haas Grotesk"
  Character: Clean, modern, direct — reads well at any size on dark backgrounds
  Use: ALL body copy, navigation, product names in grid, descriptions,
       prices, size selectors, form fields, checkout, cart metadata
  Weights: 400 (regular), 500 (medium for emphasis), 600 (semi-bold for prices/CTAs)
  Sizes:
    Navigation:    13px, letter-spacing 0.06em
    Body:          15px, line-height 1.7
    Product name:  14–16px
    Price:         18–22px, weight 600
    Caption/meta:  12px, letter-spacing 0.04em
    CTA buttons:   13–14px, letter-spacing 0.08em, all-caps

ACCENT / TECHNICAL FONT — "Space Mono" or "IBM Plex Mono"
  Use: Product spec tables (fabric %, gsm, stretch rating), size charts,
       stock status ("LAST 3 UNITS"), technical data points
  Size: 11–13px only
  Creates a "technical spec sheet" feel — signals quality and precision

TYPOGRAPHIC RULES:
  — Display font for emotional/campaign moments (hero, section headings)
  — DM Sans for all commerce and interface copy
  — Mono for technical specs and scarcity signals
  — Never more than 2 fonts visible simultaneously in one section
  — All prices in DM Sans semi-bold — never lightweight pricing
  — Product names sentence case (not all-caps) — easier to scan
```

### 2.3 Spacing System

```
BASE UNIT: 8px — all multiples of 8

4px   — micro (badge padding, icon gap)
8px   — xs (tight inline)
12px  — product grid gap (tight — gym stores feel abundant)
16px  — sm (component internal)
24px  — md (standard gap)
32px  — lg (card padding, grouped elements)
48px  — xl (section internal)
64px  — 2xl (component separation)
80px  — section padding standard
120px — section padding large  
160px — hero internal

SPATIAL PHILOSOPHY:
  This is not a luxury brand that breathes and floats.
  It is a performance brand that packs value and confidence into every scroll.
  Product grids are dense (12px gap). Trust signals are stacked efficiently.
  The only generous whitespace moments are: hero, campaign callout, brand story.
  Everywhere else: purposeful density. Show the products. Make it easy to add to bag.
```

### 2.4 Surface Rules

```
Border radius:
  Product cards:      4px (slight warmth — not brutal zero, not bubbly)
  Buttons:            4px (matches cards — consistent, clean)
  Form inputs:        4px
  Modals:             8px
  Size buttons:       4px
  Badges/tags:        4px

Borders: 1px solid #2A2A2A on all card surfaces
Product images: no border — bleed into dark background

Shadows:
  Cards on hover: 0 8px 32px rgba(0,0,0,0.4) — depth, not decoration
  Modal: 0 24px 64px rgba(0,0,0,0.6)
  Sticky nav on scroll: 0 2px 20px rgba(0,0,0,0.5)

Photography treatment:
  All product images: dark background, high contrast
  On-body shots: gym environment OR clean dark studio
  Never: white studio background on product cards
  Slight vignette on lifestyle images: subtle dark edge
  Video content: high contrast, slightly desaturated — cinematic gym feel
```
